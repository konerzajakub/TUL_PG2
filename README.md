## 📦 VCPKG Dependencies

| Knihovna          | Popis                                                                 |
|-------------------|-----------------------------------------------------------------------|
| **glew**          | **OpenGL Extension Wrangler Library** – Přístup k funkcím OpenGL a rozšířením. |
| **glfw3**         | **GLFW** – Knihovna pro vytváření oken a správa vstupů s podporou OpenGL kontextu. |
| **glm**           | **OpenGL Mathematics** – Matematická knihovna pro 3D grafiku, poskytuje vektory, matice atd.. |
| **nlohmann-json** | **JSON for Modern C++** – Knihovna pro práci s JSON daty v C++. |
| **opencv**        | **OpenCV** – Knihovna pro počítačové vidění, zpracování obrazu a strojové učení. *Doporučuje se spíš system-wide instalace kvůli velikosti (600MB+).* |
